# ChatEx

A real-time chat application developed using Node.js, Express.js, and Socket.IO. This project demonstrates WebSocket-based communication, enabling instant message exchange between connected users.

## Features
- Real-time messaging
- Instant message broadcasting
- Socket.IO integration
- HTML Only UI
- Client-server communication using WebSockets

## Tech Stack
- Node.js
- Express.js
- Socket.IO
- HTML
